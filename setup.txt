To setup the instance, here are the steps that need to be performed.

# to install octave on ubuntu
sudo add-apt-repository ppa:octave/stable
sudo apt-get update
sudo apt-get install octave

# to install pip
sudo apt-get update && sudo apt-get -y upgrade
sudo apt-get install python-pip
pip -V

# to install oct2py
pip install oct2py
